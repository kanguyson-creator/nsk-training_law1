คู่มือสั้น: ชุดเว็บประมวลกฎหมายการบัญชีสหกรณ์ NSK แบบแก้ลิงก์เองได้

ไฟล์ในชุดนี้
1) index_editable.html
   - ใช้กับ GitHub Pages / Web hosting แบบมีไฟล์หลายไฟล์
   - ต้องอัปโหลดคู่กับ laws-data.js

2) laws-data.js
   - ฐานข้อมูลรายการกฎหมายและลิงก์
   - เวลาพบลิงก์ผิด ให้แก้ URL ในไฟล์นี้ แล้ว deploy ใหม่
   - โครงสร้างข้อมูลอยู่ในรูป JSON อ่านง่าย

3) index_standalone_itopplus.html
   - ใช้กับ iTopPlus แบบวางโค้ดทั้งก้อน
   - ข้อมูลถูกฝังในไฟล์เดียว ไม่ต้องใช้ laws-data.js แยก

วิธีแก้ลิงก์บน GitHub Pages
1. เปิด repo
2. คลิกไฟล์ laws-data.js
3. กด Edit
4. ค้นชื่อกฎหมายที่ต้องการ
5. แก้ค่า "url": "..."
6. Commit changes
7. รอ 1-3 นาที แล้วกด Ctrl+F5 ที่หน้าเว็บ

วิธีแก้ผ่านหน้าเว็บ
1. เปิดเว็บ
2. กดปุ่ม ⚙️ แก้ลิงก์
3. เลือกหมวดและรายการ
4. แก้ URL
5. กด บันทึกในหน้านี้ (จะเห็นผลเฉพาะ browser นี้ก่อน)
6. กด Copy laws-data.js ใหม่ หรือ Download laws-data.js
7. เอาไฟล์/โค้ดที่ได้ไปแทนใน GitHub เพื่อให้ทุกคนเห็นถาวร

ข้อควรจำ
- เว็บ Static ไม่มีฐานข้อมูล server จึงไม่สามารถ save ให้ทุกคนเห็นทันทีโดยไม่ deploy ใหม่
- ถ้าใช้ GitHub Pages แนะนำแก้ถาวรที่ laws-data.js
- ถ้าใช้ iTopPlus แบบวางโค้ด ให้ใช้ไฟล์ index_standalone_itopplus.html
